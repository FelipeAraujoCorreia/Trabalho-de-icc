Comandos Úteis do Git HUB

O GitHub é uma plataforma de compartilhamento de código-fonte que facilita muito o trabalho de um desenvolvedor, principalmente quando se trabalha em grupo

Abaixo, listaremos alguns comandos úteis para utilizar quando for trabalhar com Git Hub .

init Cria um novo repositório Git vazio na pasta atual. git init

Como fazer a configuração e o acesso ao repositório GitHub.

Para configurar e acessar o seu repositório do GitHub em seu computador é necessário seguir alguns passos muito simples:

  1. Com o Git já instalado em sua máquina, abra o terminal e execute os seguintes comandos de configuração:

    git config --global user.name "Seu Nome"

    git config --global user.email seu-email@exemplo.com

  2. No repositório recém-criado no GitHub, clique no botão "Code" (ou "Código") verde e copie o URL do repositório. Depois disso, execute o seguinte código:

    git clone URL_DO_REPOSITÓRIO

Agora você pode criar, modificar e excluir arquivos dentro do diretório do repositório no seu computador.
